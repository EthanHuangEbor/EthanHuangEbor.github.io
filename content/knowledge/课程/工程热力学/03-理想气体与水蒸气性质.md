# 工程热力学：理想气体与水蒸气性质

工质性质决定公式能不能用。空气在常温常压、远离液化状态时常可近似为理想气体；水蒸气、制冷剂蒸气通常不能直接套理想气体公式，应查表或查图。

## 核心判断

理想气体的关键性质是：`u` 和 `h` 只与温度有关。水蒸气的关键性质是：饱和区内压力和温度一一对应，湿蒸汽还必须知道干度才能确定状态。

状态判断优先级：

- 如果题目明确是空气、氮气、氧气且远离液化状态，可考虑理想气体。
- 如果题目是锅炉、汽轮机、凝汽器、水蒸气，优先查水蒸气表。
- 如果给出湿蒸汽干度 `x`，必须用饱和表插值。

## 理想气体状态方程

常用形式：

- `pV = m R_g T`。
- `pv = R_g T`。
- `pV = n R T`。
- `R = 8.3145 J/(mol K)`。
- `R_g = R / M`。

使用状态方程时，`p` 必须是绝对压力，`T` 必须是 K，单位必须统一。很多失分来自把表压、摄氏温度、`kPa` 和 `Pa` 混在一起。

## 比热容

普通比热容随过程而变；定容、定压过程固定后，定义 `c_v` 和 `c_p`。

- `c_v = (partial u / partial T)_v`。
- `c_p = (partial h / partial T)_p`。
- 迈耶公式：`c_p - c_v = R_g`。
- 比热比：`k = c_p / c_v`。
- `c_v = R_g / (k - 1)`。
- `c_p = k R_g / (k - 1)`。

定值比热容是近似。温差较大时，更严格做法是查平均比热或积分变比热。

## 理想气体内能、焓与熵

理想气体：

- `du = c_v dT`。
- `dh = c_p dT`。
- 定值比热近似：`delta u = c_v(T2 - T1)`。
- 定值比热近似：`delta h = c_p(T2 - T1)`。
- 变比热：`delta u = integral c_v dT`。
- 变比热：`delta h = integral c_p dT`。

熵变只与初终态有关。即使实际过程不可逆，也可以选一条连接同样初终态的可逆路径来计算：

- `delta s = c_v ln(T2/T1) + R_g ln(v2/v1)`。
- `delta s = c_p ln(T2/T1) - R_g ln(p2/p1)`。
- `delta s = c_p ln(v2/v1) + c_v ln(p2/p1)`。

这些熵变公式适用于理想气体初终平衡态，不要求实际路径可逆。

## 水蒸气状态区

水蒸气有明显相变区。饱和状态下，饱和压力 `p_s` 与饱和温度 `T_s` 一一对应，只有一个独立变量。

常见状态：

- 未饱和水：`t < t_s(p)` 或 `p > p_s(t)`。
- 饱和水：`x = 0`。
- 湿饱和蒸汽：`0 < x < 1`。
- 干饱和蒸汽：`x = 1`。
- 过热蒸汽：`t > t_s(p)`。

干度定义为 `x = m_vapor / (m_liquid + m_vapor)`。干度只对湿蒸汽区有意义，过冷水和过热蒸汽没有干度。

## 湿蒸汽插值

饱和水用单撇表示，干饱和蒸汽用双撇表示。湿蒸汽参数按干度线性插值：

- `v_x = v' + x(v'' - v')`。
- `h_x = h' + x(h'' - h')`。
- `s_x = s' + x(s'' - s')`。
- `u_x = u' + x(u'' - u')`。
- 也可用 `u_x = h_x - p v_x`。

水的临界点常用数值：`p_cr = 22.064 MPa`，`t_cr = 373.99 °C`，`v_cr = 0.003106 m^3/kg`。

## 查表路径

给 `p` 和 `t` 时，先判状态区。把给定温度与该压力下饱和温度比较，或把给定压力与该温度下饱和压力比较。

饱和水/蒸汽：给 `p_s` 或 `t_s` 即可查到一组饱和参数。湿蒸汽再用干度 `x` 插值。

过热蒸汽和未饱和水：需要两个独立参数，常用 `p` 和 `t` 查表。

不要跨越饱和分界线做普通线性插值。如果点靠近分界线，应确认它仍位于同一状态区。

## 图像

`p-v` 图适合看可逆边界功。`T-s` 图适合看可逆传热和循环热效率。水蒸气图上要抓住“一点、两线、三区、五态”：临界点，饱和水线和干饱和蒸汽线，未饱和水区、湿蒸汽区、过热蒸汽区，以及未饱和水、饱和水、湿蒸汽、干饱和蒸汽、过热蒸汽。

`h-s` 图适合蒸汽动力装置分析。汽轮机近似等熵膨胀时，焓降直接估算输出功。

## 易错点

- 理想气体状态方程不能用表压和摄氏温度。
- 水蒸气通常不能当理想气体处理。
- 饱和状态下 `p_s` 和 `T_s` 不是两个独立变量。
- 干度只对湿蒸汽区有意义。
- 不可逆绝热过程不等于熵不变。
- 查表前不判状态区，很容易把过热蒸汽查成湿蒸汽。

## 题型入口

理想气体状态题：先统一单位，再用 `pv = R_gT`。

热量和功题：用 `delta u = c_v delta T`、`delta h = c_p delta T` 接入 [[工程热力学：热力学第一定律]]。

水蒸气状态题：给 `p, t` 先判状态区；给 `p, x` 或 `t, x` 用湿蒸汽插值。

汽轮机题：常用 `s2 = s1` 找出口状态，再由焓降求功。

节流题：用 `h1 = h2` 反推后态，注意熵增。
