# 工程热力学：气体动力循环

气体动力循环把真实内燃机和燃气轮机中的燃烧、排气、压缩和膨胀过程简化为理想气体循环。它是航空发动机和燃气轮机学习的入口。

## 空气标准假设

常用假设：

- 工质取空气，按理想气体处理。
- 压缩和膨胀视为可逆绝热。
- 燃烧等效为外界加热。
- 排气等效为向低温热源放热。
- 比热容常取定值。

通用热效率：

`η_t = w_net / q_in = 1 - q_out / q_in`。

其中 `k = c_p/c_v`。

## 奥托循环

奥托循环是定容加热循环，可作为汽油机理想循环。

过程：

- `1-2`：等熵压缩。
- `2-3`：定容吸热。
- `3-4`：等熵膨胀。
- `4-1`：定容放热。

压缩比：`epsilon = v1/v2`。

热量：

- `q_in = c_v(T3 - T2)`。
- `q_out = c_v(T4 - T1)`。

热效率：

`η_t = 1 - 1/epsilon^(k-1)`。

提高压缩比可提高效率，但实际汽油机受爆燃限制。

## 狄塞尔循环

狄塞尔循环是定压加热循环，可作为柴油机理想循环。

过程：

- `1-2`：等熵压缩。
- `2-3`：定压吸热。
- `3-4`：等熵膨胀。
- `4-1`：定容放热。

压缩比：`epsilon = v1/v2`。预胀比：`rho = v3/v2`。

热量：

- `q_in = c_p(T3 - T2)`。
- `q_out = c_v(T4 - T1)`。

热效率：

`η_t = 1 - (rho^k - 1) / [k epsilon^(k-1)(rho - 1)]`。

在压缩比一定时，预胀比越大，平均吸热温度变化会影响效率，不能只背结论，要看比较条件。

## 萨巴德循环

萨巴德循环也叫混合加热循环，吸热分成定容和定压两段，更接近实际柴油机燃烧过程。

过程：

- `1-2`：等熵压缩。
- `2-3`：定容吸热。
- `3-4`：定压吸热。
- `4-5`：等熵膨胀。
- `5-1`：定容放热。

参数：

- 压缩比 `epsilon = v1/v2`。
- 增压比 `lambda = p3/p2`。
- 预胀比 `rho = v4/v3`。

热量：

- `q_in = c_v(T3 - T2) + c_p(T4 - T3)`。
- `q_out = c_v(T5 - T1)`。

热效率：

`η_t = 1 - (lambda rho^k - 1) / {epsilon^(k-1)[(lambda - 1) + k lambda(rho - 1)]}`。

## 三种活塞循环比较

比较题最重要的是先写条件。

压缩比相同、吸热量相同时：

`η_奥托 > η_萨巴德 > η_狄塞尔`。

最高压力、最高温度相同时：

`η_狄塞尔 > η_萨巴德 > η_奥托`。

这两个结论不能混用。压缩比、吸热量、最高压力、最高温度代表不同工程限制。

## 布雷顿循环

燃气轮机定压加热理想循环称为布雷顿循环。它与 [[布雷顿循环]] 节点直接相连。

过程：

- `1-2`：压气机等熵压缩。
- `2-3`：燃烧室定压吸热。
- `3-4`：燃气轮机等熵膨胀。
- `4-1`：定压放热。

增压比：`pi = p2/p1`。

热量：

- `q_in = c_p(T3 - T2)`。
- `q_out = c_p(T4 - T1)`。

热效率：

`η_t = 1 - 1/pi^((k-1)/k)`。

理想布雷顿循环中，热效率随增压比提高而提高；在最高温度给定时，净功存在最佳增压比。提高涡轮前温度有利于提高比功，但受材料耐温和冷却技术限制。

## 提高效率的方法

活塞式内燃机：提高压缩比通常能提高效率，但受爆燃、机械强度、温度和排放限制。

燃气轮机：当透平排气温度高于压气机出口温度时，可用回热器预热压缩空气，减少外部供热量。

分级压缩加中间冷却、分级膨胀加中间再热，通常能增加比功。若没有回热，单独中冷或再热不一定提高热效率，甚至可能降低热效率；与回热组合时才更有利。

## 易错点

- 把压缩比 `epsilon`、增压比 `pi`、增压比 `lambda`、预胀比 `rho` 混淆。
- 奥托、狄塞尔、萨巴德比较时忘记约束条件。
- 定容吸热用错 `c_p`，定压吸热用错 `c_v`。
- 认为提高增压比总能提高布雷顿净功。最高温度受限时，净功有最佳点。
- 认为再热、中冷单独一定提高燃气轮机热效率。

## 题型入口

效率证明：写 `η = 1 - q_out/q_in`，再用过程热量和等熵关系消去温度比。

循环作图：奥托是两条等熵线和两条定容线；狄塞尔有一条定压吸热线；萨巴德有定容和定压两段吸热；布雷顿是两条等熵线和两条定压线。

比较题：先写题目约束，再比较平均吸热温度、放热温度或代入效率式。
