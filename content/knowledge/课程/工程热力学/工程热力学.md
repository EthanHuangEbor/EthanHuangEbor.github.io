# 工程热力学

工程热力学不是公式仓库，而是一套能量账本。先选系统和边界，再判断质量、热量、功量怎样穿过边界；第一定律负责数量守恒，第二定律负责方向、限度和损失。真正会做题时，看到题面能迅速判断：该用闭口系还是开口系，该查水蒸气表还是用理想气体模型，该在 `p-v` 图看功还是在 `T-s` 图看热。

## 课程总框架

这门课可以压成一条主线：

对象与描述 -> 能量守恒 -> 工质性质 -> 热力过程 -> 方向与限度 -> 循环与装置。

第一层是概念层：系统、边界、状态、平衡、准静态、可逆、热、功、循环。它决定题目从哪里切入。

第二层是方程层：闭口系第一定律、稳定流动能量方程、熵方程、克劳修斯不等式和孤立系熵增原理。它决定题目怎么算。

第三层是工程层：理想气体、水蒸气、湿空气、内燃机循环、燃气轮机循环、朗肯循环、制冷与热泵。它决定模型和近似能不能用。

## 复习路径

第一遍先学会“选系统”。看到活塞气缸、喷管、压气机、汽轮机、换热器、节流阀、锅炉、凝汽器，先判断边界有没有质量流动，有没有轴功，有没有热交换，动能位能能不能忽略。

第二遍把性质算准。理想气体题通常用 `pv = R_gT`、`du = c_vdT`、`dh = c_pdT`、熵变公式；水蒸气题先判状态区，再查表或查图，湿蒸汽用干度插值。

第三遍训练图像。`p-v` 图的面积对应可逆体积功，`T-s` 图的面积对应可逆传热。画错图，计算题经常也会跟着错。

第四遍做循环。所有循环都回到 `η = w_net / q_in = 1 - q_out / q_in` 或 `COP = 收益热量 / 输入功`，区别只在于吸热、放热、压缩和膨胀过程怎么表达。

## 考试结构

总复习课件给出的卷面结构是：简答约 40 分，作图约 20 分，证明约 20 分，计算约 20 分。闭卷，可带简单计算器。

简答题重点是概念边界。典型问法包括：准静态与可逆的关系，绝热与定熵的区别，焓为什么用于开口系，卡诺定理的意义，熵增原理为什么只适用于孤立系统。

作图题重点是过程性质和方向。要能在 `p-v`、`T-s`、`h-s` 图上画基本过程和典型循环，并说明面积意义的适用条件。

证明题重点是条件。第二定律两种表述等效、卡诺定理、克劳修斯不等式、循环效率表达式，都要先写清适用前提。

计算题重点是流程。先画状态点，再定模型，再列方程，最后检查单位、符号和物理量级。

## 快速总公式

- 闭口系第一定律：`q = delta u + w`。
- 稳定流动能量方程：`q = delta h + delta(c^2/2) + g delta z + w_s`。
- 焓定义：`h = u + pv`。
- 可逆体积功：`w = integral p dv`。
- 可逆技术功：`w_t = - integral v dp`。
- 可逆传热：`delta q_rev = T ds`。
- 热效率：`η = w_net / q_in = 1 - q_out / q_in`。
- 制冷系数：`COP_R = q_L / w_in`。
- 热泵系数：`COP_HP = q_H / w_in = COP_R + 1`。
- 孤立系熵增：`delta S_iso >= 0`。
- 㶲损：`I = T0 S_gen`。

## 和飞行动力的连接

工程热力学直接连接航空发动机和飞行器动力方向。[[布雷顿循环]] 是燃气涡轮发动机的基础热力循环，[[标准大气模型]] 把高度转成入口压力、温度和密度，[[压气机喘振]] 则提醒热力循环之外还要考虑部件稳定性。

后续可以把发动机总体设计、燃气轮机回热/再热、中冷压缩、喷管膨胀等内容继续拆成项目化节点。
