# 工程热力学：蒸汽动力循环与制冷热泵

本章部分内容按工程热力学课程框架补充。核心口径是：蒸汽动力和制冷剂循环通常不能套理想气体公式，要用水蒸气表、制冷剂表或 `T-s`、`h-s`、`p-h` 图。

## 朗肯循环

朗肯循环是简单蒸汽动力循环，以水和水蒸气为工质。

典型过程：

- `1-2`：汽轮机等熵膨胀。
- `2-3`：凝汽器定压放热。
- `3-4`：给水泵绝热压缩。
- `4-1`：锅炉定压吸热。

常用焓差：

- 汽轮机功：`w_t = h1 - h2`。
- 泵功：`w_p = h4 - h3 ≈ v3(p4 - p3)`。
- 吸热量：`q_in = h1 - h4`。
- 放热量：`q_out = h2 - h3`。
- 热效率：`η_t = [(h1 - h2) - (h4 - h3)] / (h1 - h4)`。

若忽略泵功，可近似 `η_t ≈ (h1 - h2)/(h1 - h3)`。

汽轮机出口若进入湿蒸汽区，用 `s2 = s1` 结合出口压力查饱和表求干度：

- `x2 = (s2 - s_f)/(s_g - s_f)`。
- `h2 = h_f + x2(h_g - h_f)`。

工程上希望汽轮机末端干度较高，以减少水滴冲蚀。

## 提高朗肯循环效率

基本方向：

- 提高平均吸热温度。
- 降低平均放热温度。
- 减少设备不可逆损失。

常见措施：

- 提高初温。
- 提高初压。
- 降低背压。
- 采用再热循环。
- 采用回热循环。
- 燃气-蒸汽联合循环或热电联产。

提高初压有利于效率，但可能降低末端干度。降低背压受环境温度和凝汽条件限制。

## 再热循环

再热循环把汽轮机膨胀分成两段：高压缸膨胀后回锅炉再热，再进入低压缸膨胀。

若忽略泵功，常用：

- `q_in = (h1 - h4) + (h_b - h_a)`。
- `w_net ≈ (h1 - h_a) + (h_b - h2)`。
- `η_t,RH = [(h1 - h_a) + (h_b - h2)] / [(h1 - h4) + (h_b - h_a)]`。

再热的主要目的不是保证热效率必然上升，而是提高乏汽干度，并为提高初压创造条件。再热压力选择很关键，过高或过低都会削弱收益。

## 回热循环

回热循环从汽轮机中间级抽出部分蒸汽加热给水，提高锅炉入口水温，从而提高平均吸热温度。

混合式回热器能量平衡：

- `alpha h_a + (1 - alpha)h4 = h5`。
- `alpha = (h5 - h4)/(h_a - h4)`。

回热循环中各段质量流量不同，因此 `T-s` 图面积不能简单代表热量或功量。

回热的效果：

- 提高循环热效率。
- 减少低压缸末级流量和凝汽器负担。
- 通常降低循环比功。
- 增加设备复杂性和投资。

## 制冷循环与热泵循环

逆循环消耗外界功，把热量从低温处转移到高温处。若收益是低温端吸热，称为制冷；若收益是高温端放热，称为热泵。

性能指标：

- 制冷系数：`COP_R = Q_L/W_in = q_L/(q_H - q_L)`。
- 热泵系数：`COP_HP = Q_H/W_in = q_H/(q_H - q_L)`。
- 关系：`COP_HP = COP_R + 1`。

逆卡诺极限：

- `COP_R,C = T_L/(T_H - T_L)`。
- `COP_HP,C = T_H/(T_H - T_L)`。

温度必须用 K。

## 蒸气压缩制冷循环

典型部件：压缩机、冷凝器、节流阀、蒸发器。

典型过程：

- `1-2`：压缩机压缩，输入功。
- `2-3`：冷凝器放热。
- `3-4`：节流阀节流，近似等焓。
- `4-1`：蒸发器吸热。

常用公式：

- `q_L = h1 - h4`。
- `q_H = h2 - h3`。
- `w_in = h2 - h1`。
- `COP_R = (h1 - h4)/(h2 - h1)`。
- `COP_HP = (h2 - h3)/(h2 - h1)`。
- 节流：`h3 = h4`。

提高性能的方向：

- 提高蒸发温度。
- 降低冷凝温度。
- 减少压缩机不可逆损失。
- 降低换热器压降。
- 合理利用过冷和过热。

## 易错点

- 对水蒸气或制冷剂套理想气体公式。
- 泵功单位中 `MPa`、`kPa`、`kJ/kg` 没有统一。
- 汽轮机出口干度超出 `0-1` 仍按湿蒸汽处理。
- 再热误写成一定提高热效率。
- 回热循环仍用 `T-s` 面积直接代表热量和功量。
- 热泵系数和制冷系数分子写反。
- 逆卡诺公式使用摄氏温度。

## 题型入口

朗肯计算：先在 `T-s` 或 `h-s` 图上标状态点，再查 `h, s, x`，最后用焓差求功、热和效率。

再热计算：拆成高压缸和低压缸两段焓降，吸热量也分锅炉主吸热和再热吸热。

回热计算：先用回热器能量平衡求抽汽份额，再写各段质量流量对应的功和热。

制冷计算：先定四个点，节流阀用等焓，压缩机用焓升，蒸发器吸热作为制冷量。
