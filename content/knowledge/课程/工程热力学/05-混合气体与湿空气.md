# 工程热力学：混合气体与湿空气

混合物题的核心是把总体拆成组分，再明确使用质量口径还是摩尔口径。湿空气是干空气和水蒸气组成的特殊混合气体，工程上通常按理想气体混合物处理。

## 核心判断

理想气体混合物可看成一种折合的假想理想气体：

`pV = m R_g,eq T = n R T`。

但加权方式要分清：

- 分压力、分体积、体积分数通常对应摩尔分数。
- 折合摩尔质量用摩尔分数加权。
- 折合气体常数可由 `R/M_eq` 得到，也可用质量分数加权。

湿空气判断饱和与否，只看同一温度下水蒸气分压力 `p_v` 与饱和压力 `p_s(t)` 的关系。

## 分压力

分压力定义：某组分在混合物温度 `T` 下，单独占据混合物总体积 `V` 时具有的压力。

对第 `i` 个组分：

- `p_i V = n_i R T`。
- `p = sum p_i`。
- `p_i/p = n_i/n = x_i`。

其中 `x_i` 是摩尔分数。分压力定律也叫道尔顿定律，只适用于理想气体混合物。

## 分体积

分体积定义：某组分在混合物温度 `T` 和总压力 `p` 下单独存在时占据的体积。

- `p V_i = n_i R T`。
- `V = sum V_i`。
- `V_i/V = n_i/n = x_i`。

理想气体混合物中，`p_i/p = V_i/V = x_i`。分体积定律也叫亚美格定律。

## 折合参数

折合摩尔质量：

`M_eq = sum x_i M_i`。

折合气体常数：

`R_g,eq = R / M_eq`。

也可按质量分数：

`R_g,eq = sum w_i R_i`。

质量分数与摩尔分数换算：

- `w_i = x_i M_i / M_eq`。
- `x_i = (w_i/M_i) / sum(w_j/M_j)`。
- `x_i = w_i R_i / R_g,eq`。

混合气体比热容常用质量分数加权：

- `c_p = sum w_i c_p,i`。
- `c_v = sum w_i c_v,i`。

## 湿空气状态

湿空气总压力由干空气分压力和水蒸气分压力组成：

`p = p_a + p_v`。

相对湿度：

- `phi = p_v / p_s(t)`。
- 也可理解为同温度下实际水蒸气密度与饱和水蒸气密度之比。

状态判断：

- `p_v < p_s(t)`：未饱和空气。
- `p_v = p_s(t)`：饱和空气。
- 继续冷却到 `p_v` 对应的饱和温度以下，会发生结露。

露点温度为 `t_d = t_s(p_v)`。

## 含湿量

含湿量定义为每 kg 干空气所含水蒸气质量：

`d = m_v / m_a`。

常用公式：

- `d = 0.622 p_v / (p - p_v)`。
- `d = 0.622 phi p_s(t) / (p - phi p_s(t))`。
- `p_v = p d / (0.622 + d)`。

在总压力给定时，`d`、`p_v`、露点 `t_d` 一一对应。单纯加热或冷却且无加湿除湿时，`d` 不变。

## 湿空气焓

湿空气焓通常按每 kg 干空气定义：

`h = h_a + d h_v`。

工程常用式：

`h = 1.005 t + d(2501 + 1.86 t)`。

其中 `h` 单位为 `kJ/kg 干空气`，`t` 用摄氏度，`d` 用 `kg 水蒸气/kg 干空气`。

注意这里的摄氏温度只适用于这条工程经验式，不代表理想气体状态方程也能用摄氏温度。

## h-d 图

湿空气 `h-d` 图通常在总压力给定下使用。知道任意两个独立湿空气参数即可定状态，例如 `t + phi`、`t + d`、`t + t_w`。

单纯加热过程：`d` 不变，温度升高，相对湿度下降。

单纯冷却过程：`d` 不变，温度下降，相对湿度上升；若达到露点后继续冷却，会析出水分，`d` 降低。

加湿过程和除湿过程要看是否伴随加热、冷却或喷淋，不能只凭温度变化判断。

## 易错点

- 道尔顿和亚美格定律只适用于理想气体混合物。
- 质量分数大的组分，摩尔分数不一定大。
- 水蒸气分压力不是总压力。
- 相对湿度不是含湿量。升温时若 `p_v` 不变，`d` 不变但 `phi` 会下降。
- 湿空气焓的基准是 kg 干空气，不是 kg 湿空气。
- 结露判断看壁面温度是否低于露点。

## 题型入口

混合气体成分换算：先把质量分数和摩尔分数统一，不要直接比较。

分压力/分体积题：用 `p_i = x_i p`、`V_i = x_i V`。

折合参数题：求 `M_eq` 用摩尔分数加权，求 `R_g,eq` 用 `R/M_eq` 或质量分数加权。

湿空气题：给 `p, t, phi`，先查 `p_s(t)`，再算 `p_v`、`d`、`t_d`、`h`。

结露题：求露点；壁面温度低于露点则凝结。
