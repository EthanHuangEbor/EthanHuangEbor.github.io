# 工程热力学：基本概念

本章决定整门课的入口。热力学题最怕一上来套公式，因为公式的适用对象取决于系统怎么选、边界怎么画、过程能否看成平衡态序列。

## 核心判断

拿到题先问四件事：

- 系统是谁，边界在哪里。
- 边界上有没有质量穿过。
- 边界上有没有热量和功量交换。
- 过程能不能用一串平衡状态描述，能不能进一步看成可逆。

如果系统和边界选错，后面的 `Q`、`W`、`m h` 都会错。

## 系统、边界与外界

热力系统是人为选定的研究对象。外界是系统之外、能与系统发生相互作用的部分。边界是系统与外界之间的分界面，可以是真实的，也可以是假想的；可以固定，也可以移动；可以允许质量通过，也可以不允许。

常见分类：

- 闭口系：没有质量穿过边界，可以有热和功交换。
- 开口系：有质量穿过边界，也可能有热和功交换。
- 绝热系：没有热量交换，但可能有功或质量交换。
- 孤立系：没有质量、热量、功量交换。
- 简单可压缩系：通常只考虑热量和体积变化功，不考虑电、磁、表面张力等复杂作用。

一个装置用什么系统取法，取决于问题目标。例如压气机通常取控制体，是开口系；活塞气缸内固定质量气体通常取闭口系；自由膨胀若把整个刚性绝热容器内气体作为系统，则 `Q = 0` 且 `W = 0`。

## 状态与状态参数

状态是系统某一瞬间的宏观物理状况。状态参数只由状态决定，与到达该状态的路径无关。

常用状态参数：

- 基本状态参数：`p, v, T`。
- 能量类状态参数：`u, h`。
- 第二定律状态参数：`s`。
- 广延量：`V, U, H, S, m`，与质量多少有关。
- 强度量：`p, T, v, u, h, s`，与系统总质量无关。

比参数是单位质量的广延量，例如 `v = V/m`、`u = U/m`、`h = H/m`、`s = S/m`。工程热力学多数公式默认使用单位质量形式。

## 压力与温度

理想气体状态方程、水蒸气表和循环公式中的压力通常是绝对压力，不是表压。若环境压力为 `p_b`：

- 压力高于环境时：`p = p_b + p_g`。
- 压力低于环境时：`p = p_b - p_v`。

热力学温度必须用 K：`T/K = t/°C + 273.15`。卡诺效率、理想气体温比、熵变公式都不能直接代入摄氏温度。

## 平衡状态

平衡状态要求系统内部及系统与外界之间不存在推动宏观变化的势差。常见平衡包括热平衡、力平衡、相平衡和化学平衡。

平衡、稳定、均匀不能混同：

- 稳定只说明宏观参数不随时间变，可能依赖外界维持。
- 平衡强调没有自发变化趋势。
- 均匀强调空间分布一致，平衡强调势差消失。
- 单相平衡通常均匀，多相平衡不一定处处均匀。

状态点只有在平衡态或近似平衡态下才有明确意义。

## 准静态过程

准静态过程是过程足够缓慢，系统在每一瞬间都近似处于平衡状态。它是实际过程的理想化，允许我们在状态图上画连续曲线。

判断准静态时，可以看外界作用变化是否远慢于系统内部恢复平衡的时间。活塞缓慢移动时可以近似准静态；爆炸、节流、自由膨胀、剧烈混合通常不能看作准静态。

准静态过程能用 `p-v` 图上的状态线描述，但这还不等于可逆。

## 可逆过程

可逆过程要求过程反向进行时，系统和外界都能完全恢复原状，不留下任何影响。它必须同时满足：

- 准静态。
- 无耗散。

耗散包括摩擦、有限温差传热、节流、自由膨胀、混合、非弹性变形、电阻发热等。

重要判断：可逆过程一定准静态；准静态过程不一定可逆。可逆绝热才是定熵，不可逆绝热通常熵增。

## 功与热

功是系统与外界之间通过宏观有序运动传递的能量。热量是由温差推动、通过微观无序运动传递的能量。二者都是过程量，不是状态量。

常用符号约定：

- 系统吸热 `Q > 0`。
- 系统对外做功 `W > 0`。

简单可压缩系的可逆体积功为 `w = integral p dv`。这个公式只适用于准静态边界功。自由膨胀虽然体积变大，但边界不对外推动有效外界压力，不能直接用 `integral p dv` 算功。

可逆传热满足 `delta q_rev = T ds`。它是熵的定义入口，不是不可逆热量的直接计算式。

## 循环

循环是系统经历一系列过程后回到初态。循环中状态量总变化为零，但净热量和净功通常不为零。

- 正循环输出净功，对应热机、燃气轮机、蒸汽动力装置。
- 逆循环消耗功，把热量从低温处转移到高温处，对应制冷机和热泵。

正循环热效率为 `η = W_net / Q_in = 1 - Q_out / Q_in`。逆循环要看收益端：制冷系数 `COP_R = Q_L / W`，热泵系数 `COP_HP = Q_H / W`。

## 易错点

- 开口系可以绝热，绝热不等于无质量交换。
- 绝对压力才是状态参数，表压和真空度不能直接代入状态方程。
- 稳定不一定平衡，均匀不一定说明过程可逆。
- 准静态只说明接近平衡，不排除摩擦和有限温差。
- 功和热不是系统“含有”的能量，只在穿过边界时有意义。
- `p-v` 面积表示可逆体积功，`T-s` 面积表示可逆传热，都有前提。

## 题型入口

系统分类题：先看质量是否穿过边界，再看热和功。

压力换算题：先分清绝对压力、表压、真空度和环境压力。

状态参数题：判断是否只由初终态决定，是否可加，是否是单位质量量。

准静态/可逆题：先判能否画连续状态线，再找耗散因素。

循环题：先判正循环还是逆循环，再选热效率、制冷系数或热泵系数。
