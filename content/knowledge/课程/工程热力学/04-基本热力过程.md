# 工程热力学：基本热力过程

本章默认口径通常是理想气体、可逆过程、比热容取定值。若工质是水蒸气或制冷剂，不能直接套理想气体公式，应先由蒸汽表或图确定 `u, h, s, v, x`，再用能量方程计算。

## 核心判断

热力过程题先抓四件事：

- 过程方程：定容、定压、定温、定熵还是多变。
- 状态关系：用过程方程联立 `pv = R_gT`。
- 能量方程：闭口系用 `q = delta u + w`，开口稳流用 `q = delta h + w_t`。
- 图像方向：在 `p-v` 图看功，在 `T-s` 图看可逆传热。

快捷判断：

- `T2 > T1`，则理想气体 `delta u > 0`、`delta h > 0`。
- `v2 > v1`，则可逆膨胀功 `w > 0`。
- `p2 < p1`，则可逆技术功 `w_t > 0`。
- 可逆过程 `s2 > s1`，则吸热 `q > 0`。

## 定容过程

定容过程 `v = const`。状态关系为 `p2/p1 = T2/T1`。

闭口系体积功为零：`w = 0`。热量全部用于改变内能：`q = c_v(T2 - T1)`。

稳定流动可逆技术功为 `w_t = -v(p2 - p1)`。压力升高时技术功为负，表示需要外界输入。

图像上，`p-v` 图是竖直线，面积为零；`T-s` 图上温度升高时向右上走。

## 定压过程

定压过程 `p = const`。状态关系为 `v2/v1 = T2/T1`。

闭口系功为 `w = p(v2 - v1) = R_g(T2 - T1)`。

热量为 `q = c_p(T2 - T1)`，因为 `q = delta u + w = c_v delta T + R_g delta T`。

稳定流动可逆技术功为零：`w_t = 0`。

图像上，`p-v` 图是水平线；`T-s` 图上定压线比定容线更平缓，因为 `c_p > c_v`。

## 定温过程

理想气体定温过程 `T = const`，所以 `p1v1 = p2v2`。

理想气体内能和焓只与温度有关，因此 `delta u = 0`、`delta h = 0`。

可逆定温功：

- `w = R_g T ln(v2/v1)`。
- `w = R_g T ln(p1/p2)`。
- `w_t = w`。

闭口系第一定律给出 `q = w`。膨胀时吸热并对外做功，压缩时放热并消耗外界功。

注意：定温过程对理想气体才有 `delta u = delta h = 0`。实际气体或水蒸气不能这样套。

## 定熵过程

可逆绝热过程是定熵过程，`s = const`。理想气体定熵关系：

- `p v^k = const`。
- `T2/T1 = (v1/v2)^(k-1)`。
- `T2/T1 = (p2/p1)^((k-1)/k)`。

因为 `q = 0`，闭口系有 `w = -delta u = c_v(T1 - T2)`。稳定流动可逆绝热过程有 `w_t = -delta h = c_p(T1 - T2)`。

图像上，`T-s` 图是竖直线。`p-v` 图上定熵线比定温线更陡。

重要边界：绝热不一定定熵。只有可逆绝热才定熵；不可逆绝热通常 `delta s > 0`。

## 多变过程

多变过程统一表示为 `p v^n = const`。

特殊值：

- `n = 0`：定压。
- `n = 1`：定温。
- `n = k`：定熵。
- `n -> infinity`：定容。

常用状态关系：

- `p2/p1 = (v1/v2)^n`。
- `T2/T1 = (v1/v2)^(n-1)`。
- `T2/T1 = (p2/p1)^((n-1)/n)`。

当 `n != 1`：

- `w = (p2v2 - p1v1) / (1 - n)`。
- `w = R_g(T2 - T1) / (1 - n)`。
- `w_t = n w`。
- `q = c_n(T2 - T1)`。
- `c_n = c_v (n - k) / (n - 1)`。

当 `n = 1`，必须回到定温对数公式，不能用含 `1 - n` 的式子。

## 图像判断

在 `p-v` 图上，可逆过程曲线下面积代表膨胀功 `w`。同一初态向右膨胀时，定压线最平，定温线下降，定熵线比定温线更陡。斜率关系为 `dp/dv = -n p/v`，`n` 越大曲线越陡。

在 `T-s` 图上，可逆过程曲线下面积代表热量 `q`。定熵线竖直，定温线水平。定压线和定容线随温度升高向右上走，且定压线更平。

几个分界：

- 等温线是理想气体 `delta u`、`delta h` 正负的分界。
- 等容线是体积功 `w` 的分界。
- 等压线是技术功 `w_t` 的分界。
- 等熵线是可逆热量 `q` 的分界。

## 易错点

- 定容过程误用 `c_p`，定压过程误用 `c_v`。
- 多变指数 `n = 1` 时仍套 `w = R_g delta T/(1-n)`。
- 把绝热直接等同于定熵。
- 把体积功 `w` 和技术功 `w_t` 混用。
- 对水蒸气和制冷剂套理想气体过程公式。
- 图像面积不写可逆条件。

## 题型入口

已知初态和过程类型求终态：先写过程方程，再联立状态方程。

求热量和功：闭口系用 `q = delta u + w`；开口稳流用 `q = delta h + w_t`。

过程比较题：先比较曲线位置和面积，再转成压缩功、膨胀功或吸放热。

作图题：先判膨胀/压缩，再判升温/降温，再判吸热/放热。
