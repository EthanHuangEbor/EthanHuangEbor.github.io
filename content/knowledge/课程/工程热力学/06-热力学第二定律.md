# 工程热力学：热力学第二定律

第一定律管能量数量，第二定律管过程方向、限度和损失。满足能量守恒的过程不一定能发生，因为热过程有方向性。

## 核心问题

自发过程通常沿能量品质降低方向进行：功变热，热从高温传到低温，气体自由膨胀，不同气体混合。逆过程并非绝对不能发生，而是不能无代价自动发生，必须有补偿。

第二定律题目要问：

- 过程是否可能。
- 若可能，是可逆还是不可逆。
- 理论最大效率或最小耗功是多少。
- 不可逆损失有多大。

## 两种表述

克劳修斯表述：不可能把热量从低温物体传到高温物体而不引起其他变化。它针对热量传递方向。

开尔文表述：不可能从单一热源取热并使之完全变为功而不引起其他变化。它针对热功转换方向，也说明第二类永动机不可能制成。

两种表述等效。证明常用反证法：若能违反开尔文表述，就可用单热源热机驱动制冷机，造成热量无偿从低温传到高温，违反克劳修斯表述；反过来，若热量能无偿从低温传到高温，再配合普通热机，就能从单一热源取热并全部变成功。

## 卡诺循环

卡诺循环是在两个恒温热源 `T_H` 和 `T_L` 之间工作的理想可逆正循环，由两个可逆定温过程和两个可逆绝热过程组成。

正向卡诺热机：

- `q_H = T_H delta s`。
- `q_L = T_L delta s`。
- `w_net = q_H - q_L`。
- `η_C = w_net/q_H = 1 - q_L/q_H = 1 - T_L/T_H`。

结论：

- 卡诺效率只取决于两个热源温度，与工质无关。
- `T_H` 越高、`T_L` 越低，理论热效率越高。
- 热机循环必须放热，否则违反开尔文表述。
- 实际循环达不到卡诺循环，因为实际过程存在不可逆性。

逆卡诺循环：

- 制冷系数 `COP_R,C = T_L/(T_H - T_L)`。
- 热泵系数 `COP_HP,C = T_H/(T_H - T_L) = COP_R,C + 1`。

温度必须用 K。

## 卡诺定理

在同一高温热源和同一低温热源之间工作的一切可逆热机，热效率相等。在同一对热源之间工作的不可逆热机，热效率必小于可逆热机：

`η_irreversible < η_reversible = η_C`。

比较热机效率时，必须限定同一对热源或相同温度界限。不能把任意不可逆热机与任意可逆热机直接比较。

## 熵的定义

对可逆循环：

`cycle integral delta Q_rev/T = 0`。

因此 `delta Q_rev/T` 可以定义状态函数熵的微分：

- `dS = delta Q_rev/T`。
- 单位质量形式：`ds = delta q_rev/T`。

任意两平衡态之间的熵变只与初终态有关：

`delta S = integral_1^2 delta Q_rev/T`。

即使实际过程不可逆，也可以选择一条连接相同初终态的可逆路径来算熵变。

可逆过程中：

- `ds > 0` 对应吸热。
- `ds < 0` 对应放热。
- `ds = 0` 对应可逆绝热，即定熵。

功不携带熵。热量和质量流动可以携带熵。

## 克劳修斯不等式

对任意循环：

`cycle integral delta Q/T <= 0`。

判据：

- 等于 0：循环可行且可逆。
- 小于 0：循环可行但不可逆。
- 大于 0：循环不可能。

这里热量正负通常站在工质角度，工质吸热为正，放热为负；温度取与工质交换热量的热源温度。

对任意过程：

- `dS >= delta Q/T`。
- `delta S >= integral delta Q/T`。

等号对应可逆过程，不等号对应不可逆过程。不可逆绝热过程因为 `delta Q = 0`，所以 `delta S > 0`。

## 熵方程

闭口系熵方程：

- `dS = delta S_f,Q + delta S_g`。
- `delta S_f,Q = delta Q/T_boundary`。
- `delta S_g >= 0`。

积分形式：

`delta S = S_f,Q + S_g`。

其中 `S_f,Q` 是热熵流，可正、可负、可为零；`S_g` 是熵产，只能大于或等于零。可逆过程 `S_g = 0`，不可逆过程 `S_g > 0`。

绝热闭口系：

`delta S = S_g >= 0`。

开口系熵方程：

`dS_cv = sum(s_in delta m_in) - sum(s_out delta m_out) + delta S_f,Q + delta S_g`。

稳定流动一进一出时：

`s2 - s1 = s_f,Q + s_g`。

绝热稳定流动时：

`s2 - s1 = s_g >= 0`。

## 孤立系统熵增原理

孤立系统无质量交换、无热量交换、无功量交换，因此没有熵流，只有熵产：

`delta S_iso = delta S_g >= 0`。

判据：

- `delta S_iso > 0`：过程可行且不可逆。
- `delta S_iso = 0`：过程可逆，达到极限。
- `delta S_iso < 0`：过程不可能。

熵增原理只适用于孤立系统。非孤立系统或孤立系统中的某个子系统，熵可以增大、减小或不变。计算时常把研究对象、热源、冷源、功源和环境合并成孤立系统。

典型热传递：

`delta S_iso = -Q/T_H + Q/T_L = Q(1/T_L - 1/T_H)`。

当 `T_H > T_L` 时，`delta S_iso > 0`，热量可自发从高温传到低温。

## 㶲与不可逆损失

㶲表示在给定环境条件下，能量中最多能转化为有用功的部分。环境温度记为 `T0`。

能量可写成：`E = Ex + An`。其中 `Ex` 是㶲，`An` 是不可用能。

恒温热源 `T` 下热量 `Q` 的热量㶲：

`Ex_Q = Q(1 - T0/T)`。

变温热源：

`Ex_Q = integral (1 - T0/T) delta Q = Q - T0 delta S`。

若 `T = T0`，热量㶲为零。环境热能不能转化为有用功。

不可逆损失：

- `I = W_rev - W_actual`。
- `I = T0 S_g = T0 delta S_iso`。

这说明不可逆因素一方面造成熵产，另一方面造成做功能力损失。

## 证明题思路

两种表述等效：

- 用反证法。
- 假设某一表述被违反。
- 借助热机或制冷机组合出违反另一表述的装置。
- 合并装置后只看外界总效果，不纠结内部设备。

卡诺定理：

- 设两台热机在同一 `T_H`、`T_L` 间工作。
- 假设某台可逆机效率更高。
- 让效率较低者逆向运行，由效率较高者供功。
- 若合并后产生单热源取热并输出净功，则违反第二定律。
- 交换角色排除另一种大小关系，可逆机效率只能相等。

克劳修斯不等式：

- 先证明可逆循环 `cycle integral delta Q_rev/T = 0`。
- 再用不可逆循环与同热源间可逆循环比较。
- 借助卡诺定理说明不可逆循环效率低于可逆循环。
- 合并得到 `cycle integral delta Q/T <= 0`。

## 判断题易错点

- “任何过程熵只增不减”是错的，只有孤立系统总熵不能减小。
- 熵变只由初终态决定，同一初终态下可逆与不可逆路径的 `delta S` 相同。
- 循环工质回到初态，工质 `delta S = 0`；增大的是孤立系统总熵或熵产。
- 只有可逆绝热才定熵。
- 功不携带熵。
- “热不能全部变成功”要加循环装置和单一热源等条件。
- 克劳修斯不等式中热量正负按工质判断。

## 计算题模板

判断循环是否可行：

- 第一律检查：`W = Q_H - Q_L`。
- 卡诺判据：`η = W/Q_H`，比较 `η_C = 1 - T_L/T_H`。
- 克劳修斯判据：算 `Q_H/T_H - Q_L/T_L`。
- 孤立系判据：算热源、冷源、工质、功源总熵变。

求熵变：

- 理想气体：`delta s = c_v ln(T2/T1) + R_g ln(v2/v1)`。
- 理想气体：`delta s = c_p ln(T2/T1) - R_g ln(p2/p1)`。
- 固体或液体近似：`delta S = mc ln(T2/T1)`。
- 热源：`delta S_res = Q_res/T`。
- 循环工质：`delta S_cycle = 0`。

熵产：

- 闭口系：`S_g = delta S - integral delta Q/T_boundary`。
- 绝热闭口系：`S_g = delta S`。
- 绝热稳定流动：`s_g = s2 - s1`。

㶲损：

- 先算总熵产或孤立系统总熵增。
- 再用 `I = T0 S_g` 或 `I = T0 delta S_iso`。
