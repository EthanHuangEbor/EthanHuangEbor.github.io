# 工程热力学：热力学第一定律

第一定律回答“能量账是否平衡”。它不判断过程能不能自发发生，也不判断效率极限；这些属于 [[工程热力学：热力学第二定律]]。

## 核心判断

第一步永远是选系统。

如果是固定质量气体，常取闭口系，用 `q = delta u + w`。如果是喷管、压气机、汽轮机、换热器、节流阀，常取控制体，用稳定流动能量方程。系统边界一变，热量、功量、质量携带能量都会变。

## 总能量

系统总能量包括内能、动能和位能：

- 总量形式：`E = U + m c^2/2 + mgz`。
- 单位质量形式：`e = u + c^2/2 + gz`。

工程热力学中，很多封闭容器和普通换热设备的动能、位能变化可忽略；喷管、扩压管、叶片通道中动能变化不能随手忽略。

## 闭口系第一定律

闭口系没有质量穿过边界。常用关系：

- 微分形式：`delta Q = dU + delta W`。
- 单位质量：`q = delta u + w`。
- 循环过程：`cycle integral dU = 0`，所以 `Q_net = W_net`。

若过程是准静态简单可压缩过程，体积功为 `w = integral p dv`。若过程不可准静态，例如自由膨胀，不能把系统内部瞬时压力直接代入这个积分。

理想气体闭口系还常用：

- `delta u = c_v delta T`。
- 定值比热容近似：`delta u = c_v(T2 - T1)`。

## 焓的来源

开口系有质量进出。流动工质进入控制体时，除了携带内能 `u`，还需要外界推动它进入控制体，这部分推动功为 `pv`。因此定义焓：

- `h = u + pv`。
- `H = U + pV`。

焓不是一种神秘新能量，而是开口系能量账本中自然出现的组合量。锅炉、汽轮机、压气机、节流阀、换热器等设备，很多热量和功量都可以直接写成焓差。

## 稳定流动能量方程

一进一出、稳定流动时，取 `w_s` 为系统对外输出轴功：

`q = delta h + delta(c^2/2) + g delta z + w_s`。

若动能、位能可忽略：

`q = delta h + w_s`。

典型简化：

- 汽轮机：近似绝热，动位能忽略，`w_s ≈ h1 - h2`。
- 压气机/水泵：近似绝热，输入功大小 `w_in ≈ h2 - h1`。
- 换热器：整体绝热且无轴功时，冷热流体焓流守恒。
- 节流阀：绝热、无轴功、动位能小，`h1 = h2`。
- 喷管：近似绝热、无轴功，焓降转化为动能增加。
- 扩压管：速度降低，部分动能转化为压力升高。

## 技术功

技术功用于稳定流动可逆过程分析。可逆稳定流动满足：

- `delta q = dh + delta w_t`。
- `delta w_t = -v dp`。
- `delta q = dh - v dp = T ds`。

技术功与体积功、轴功的关系：

- `w_t = w + delta(pv)`。
- `w_t = w_s + delta(c^2/2) + g delta z`。

直观判断：压力降低时，`- integral v dp` 为正，设备有输出技术功；压力升高时，需要输入技术功。

## 非稳定流动

充放气、罐体灌装、快速排气属于非稳定开口系。不能直接套一进一出稳定流动方程。

常见刚性绝热储气罐充气，若入口状态焓为常数：

`m u_final - m0 u0 = (m - m0) h_in`。

若原罐为空：

`u_final = h_in`。

这个结论说明：进入罐内的气体不仅带来内能，还带来推动功，最终都转化为罐内气体内能。

## 典型设备速记

汽轮机和燃气轮机：焓降出功，核心式 `w_s = h1 - h2`。

压气机、泵、压缩机：焓升耗功，核心式 `w_in = h2 - h1`。

锅炉、冷凝器、换热器：热量主要对应焓差，核心式 `q = h2 - h1` 或焓流守恒。

节流阀：前后焓相等，过程强烈不可逆，不能说内部处处平衡。

喷管：焓降变速度，动能项必须保留。

## 易错点

- 不要先套公式，先选系统。
- `p dV` 只适用于准静态边界功。
- 焓用于开口流动账本，不是所有题都必须用焓。
- 节流是前后平衡截面焓相等，不是可逆等焓线。
- 喷管、扩压管不能忽略动能项。
- 闭口系没有质量流动，不应写入口出口焓流。

## 题型入口

闭口系计算：选系统，写 `q = delta u + w`，若可逆再补 `w = integral p dv`。

稳定流动设备：选控制体，写 `q = delta h + delta(c^2/2) + g delta z + w_s`，再按设备删项。

节流题：直接从 `h1 = h2` 找前后状态，但要记得熵通常增加。

充放气题：建立非稳态控制体能量守恒，不要误套稳定流动式。
