# 工程热力学：作图与计算题模板

这篇是考前入口。前面章节解决“是什么”和“怎么算”，这里把题型动作固定下来。

## 作图题模板

先判断对象：理想气体、水蒸气、制冷剂，还是完整循环。

再选图：

- 气体过程常用 `p-v`、`T-s`。
- 蒸汽动力常用 `T-s`、`h-s`。
- 制冷循环常用 `p-h` 或 `log p-h`。

作图步骤：

- 标坐标轴、状态点和过程方向。
- 标出过程性质：定容、定压、定温、定熵、多变、节流、吸热、放热。
- 正循环通常顺时针，输出净功。
- 逆循环通常逆时针，消耗净功。
- 理想气体 `p-v` 图中，定容线竖直，定压线水平，定熵线比定温线更陡。
- `T-s` 图中，定熵过程竖直；可逆吸热通常向右，放热通常向左。
- 水蒸气图先画饱和液线、干饱和蒸汽线和湿蒸汽区，再放状态点。

面积意义：

- `p-v` 图可逆过程线下面积代表体积功。
- `T-s` 图可逆过程线下面积代表热量。
- 回热循环各段质量流量不同，不能简单用 `T-s` 面积代表总热量或总功。

## 常见循环作图

奥托循环：两条等熵线，两条定容线。

狄塞尔循环：两条等熵线，一条定压吸热线，一条定容放热线。

萨巴德循环：等熵压缩、定容吸热、定压吸热、等熵膨胀、定容放热。

布雷顿循环：两条等熵线，两条定压线。

朗肯循环：汽轮机和泵近似等熵，锅炉和凝汽器为定压换热。

蒸气压缩制冷循环：压缩机升压升焓，冷凝器放热，节流阀等焓，蒸发器吸热。

## 证明题模板

证明题先写适用条件，再写核心方程，最后得出结论。

效率公式证明：

- 写 `η_t = 1 - q_out/q_in`。
- 根据过程性质写 `q_in`、`q_out`。
- 用等熵关系或焓值关系消去中间状态。
- 化为题目要求的参数形式。

循环比较证明：

- 先说明比较条件，例如“压缩比相同且吸热量相同”。
- 在图上比较平均吸热温度和平均放热温度，或比较吸放热面积。
- 用 `η_t = 1 - q_out/q_in` 得出大小关系。

第二定律证明：

- 明确孤立系统边界。
- 写熵平衡：`delta S = S_f + S_g`。
- 用 `S_g >= 0` 判断可逆、不可逆和不可能。
- 热源熵变用 `delta S_res = Q_res/T`，注意热量正负以热源为对象。

克劳修斯不等式：

`cycle integral delta Q/T <= 0`。

等号表示可逆循环，小于零表示不可逆但可行，大于零表示不可行。

## 计算题通用流程

第一步：画简图或循环图，标清状态点。

第二步：列已知量、未知量和单位，温度先换成 K。

第三步：判断模型。理想气体用状态方程和过程方程；水蒸气、制冷剂查表或查图。

第四步：逐点求状态参数：`p, v, T, h, s, x`。

第五步：对系统、设备或循环写能量方程。

第六步：求 `q`、`w`、`η_t`、`COP`、耗汽率或㶲损。

第七步：检查量级、符号、单位和物理合理性。

## 气体循环计算

等熵关系：

- `T2/T1 = (v1/v2)^(k-1)`。
- `T2/T1 = (p2/p1)^((k-1)/k)`。

热量：

- 定容热量 `q = c_v delta T`。
- 定压热量 `q = c_p delta T`。
- 净功 `w_net = q_in - q_out`。

效率：

- 奥托：`η = 1 - 1/epsilon^(k-1)`。
- 布雷顿：`η = 1 - 1/pi^((k-1)/k)`。

## 蒸汽动力计算

汽轮机理想膨胀用 `s2 = s1`。

湿蒸汽区先求干度：

- `x = (s - s_f)/(s_g - s_f)`。
- `h = h_f + x(h_g - h_f)`。

泵功：

`w_p ≈ v delta p`。

热效率优先用焓差表达，不要把水蒸气当理想气体。

## 制冷热泵计算

先在 `p-h` 图或表上确定四个点。

- 压缩机：`w_in = h2 - h1`。
- 冷凝器：`q_H = h2 - h3`。
- 节流阀：`h3 = h4`。
- 蒸发器：`q_L = h1 - h4`。
- 制冷系数：`COP_R = q_L/w_in`。
- 热泵系数：`COP_HP = q_H/w_in`。

## 第二定律计算

判断热机可行：

- 第一律：`W = Q_H - Q_L`。
- 卡诺：`η <= 1 - T_L/T_H`。
- 克劳修斯：`Q_H/T_H - Q_L/T_L <= 0`。

求熵产：

- 闭口系：`S_g = delta S - integral delta Q/T_boundary`。
- 绝热闭口系：`S_g = delta S`。
- 绝热稳定流动：`s_g = s2 - s1`。

求㶲损：

`I = T0 S_g`。

## 常见失分点

- 摄氏温度直接代入卡诺效率或理想气体温比。
- 表压直接代入状态方程。
- `epsilon`、`pi`、`lambda`、`rho` 混淆。
- 定容过程误用 `c_p`，定压过程误用 `c_v`。
- 对水蒸气和制冷剂套理想气体公式。
- 干度 `x` 超出 `0-1` 仍按湿蒸汽处理。
- 回热循环仍用 `T-s` 面积直接代表总热量和总功。
- 再热循环误写成一定提高热效率。
- 没有回热时，误认为中冷或再热一定提高燃气轮机热效率。
- 热泵系数和制冷系数分子写反。
- `MPa`、`kPa`、`kJ/kg`、`m^3/kg` 单位没有统一。

## 最后 30 分钟检查

- 公式中的温度是不是 K。
- 压力是不是绝对压力。
- 系统是闭口还是开口。
- 水蒸气是否先判状态区。
- 节流是否记住 `h1 = h2`。
- 可逆绝热和绝热是否混同。
- 画图是否标了方向。
- 结论是否写了适用条件。
