# 军事理论期末刷题记录

[打开军事理论刷题平台](/assets/quizzes/military-theory-final-review.html)

[打开军理提纲 PDF](/assets/materials/military-theory/military-theory-outline.pdf)

这个节点承接 `2026 年春季学期军理期末提纲` 和刷题平台。平台已拆分为独立题库与前端逻辑，题库目前覆盖 105 题，其中选择题 64、判断题 27、论述题 14。

## 平台结构

- 刷题：按章节、题型、难度、错题/收藏/未答范围筛选，支持顺序或随机组题。
- 模拟卷：自动抽取选择、判断和论述题，适合考前自测。
- 复习地图：按提纲章节浏览知识点，并标记掌握状态。
- 错题本：汇总错题、收藏题和章节掌握度，支持导出复盘 Markdown。

## 复习路径

1. 先用复习地图扫完整体提纲，标记不熟条目。
2. 用刷题模式按章节专项练习，错题会自动进入错题本。
3. 考前生成一套模拟卷，完成后导出复盘 Markdown。
4. 将导出的错题复盘整理成新的知识图谱节点，链接回本页。
