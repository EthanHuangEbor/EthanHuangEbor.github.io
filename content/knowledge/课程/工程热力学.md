# 工程热力学

工程热力学的核心不是公式堆叠，而是能量账本：先选系统和边界，再判断状态量、过程量、能量跨边界的形式，最后用第一定律和第二定律检查方向、限度与损失。

## 总框架

工程热力学可以压缩成三层：

1. 一般概念：系统、边界、平衡、准静态、可逆、状态量、过程量、热、功、循环。
2. 两条定律：第一定律解决“能量守恒与转换”，第二定律解决“方向、条件、限度和损失”。
3. 两类工质与循环：理想气体、水蒸气、混合气体，以及内燃机、燃气轮机、蒸汽动力循环。

看到题目先问：

1. 选闭口系还是开口系。
2. 工质是理想气体、水蒸气、湿空气还是混合气。
3. 过程是否可逆、是否绝热、是否稳定流动。
4. 需要用 `u` 还是 `h` 做能量账本。
5. 结果要判断功、热、熵变、效率还是损失。

## 第一章：基本概念

### 系统与边界

热力系统是人为分割出来的分析对象；环境是系统之外与系统发生质能交换的部分；边界是系统与环境的分界面，可以是真实或假想、固定或运动、开口或封闭。

常见系统：

- 闭口系：不交换质量，可以交换热和功。
- 开口系：可以交换质量，也可以交换热和功。
- 绝热系：无热交换。
- 孤立系：不与外界交换质量、热量和功。
- 简单可压缩系：通常只需要两个独立状态参数确定状态。

### 状态参数与过程量

状态参数是点函数，变化量与路径无关，只由初终态决定。常用状态参数有 `p, v, T, u, h, s`，其中基本状态参数是 `p, v, T`。

过程量与路径有关，典型代表是热量 `Q` 和功 `W`。热力学中常用符号约定：

- 外界向系统传热为正。
- 系统对外做功为正。

压力关系要分清绝对压力、表压力、真空度和环境压力：

- 若系统压力高于环境压力：`p = p_b + p_g`。
- 若系统压力低于环境压力：`p = p_b - p_v`。

### 平衡、准静态与可逆

平衡状态的本质是内外一切势差趋于零。平衡不一定均匀，但单相平衡态一定均匀；平衡一定稳定，稳定未必平衡。

准静态过程是系统在变化中随时近似处于平衡状态。可逆过程要求：

```text
可逆过程 = 准静态过程 + 无耗散效应
```

这句话是判断题和简答题的高频核心。

### 功、热和循环

可逆体积功：

```text
dw = p dv
w = integral p dv
```

可逆技术功：

```text
dw_t = -v dp
w_t = - integral v dp
```

可逆热量：

```text
dq_rev = T ds
q_rev = integral T ds
```

热力循环是工质经历一系列状态后回到原态的过程。正循环用于热功转换，逆循环用于制冷和热泵。

## 第二章：热力学第一定律

第一定律的实质是能量守恒与转换。一般关系是：

```text
进入系统的能量 - 离开系统的能量 = 系统能量增量
```

### 热力学能、总能与焓

热力学能 `U` 是系统内部微观粒子能量的总和。总能还包括宏观动能和位能：

```text
E = U + m c_f^2 / 2 + m g z
```

焓定义：

```text
h = u + p v
H = U + p V
```

焓是状态量。对流动工质，焓代表随工质流动携带的能量，即热力学能与推动功之和；对静止工质，焓不直接代表实际能量。

### 闭口系能量方程

闭口系任意过程：

```text
delta q = du + delta w
q = delta u + w
```

闭口系可逆过程：

```text
delta q = du + p dv = T ds
```

循环过程 `delta u = 0`，所以净热量等于净功：

```text
Q_net = W_net
```

### 稳定流动能量方程

稳定流动任意过程：

```text
q = delta h + delta(c_f^2)/2 + g delta z + w_s
```

若忽略动能和位能变化：

```text
q = delta h + w_s
```

稳定流动可逆过程还可写成：

```text
q = delta h + w_t
q = integral T ds
w_t = - integral v dp
```

典型设备快速判断：

- 换热器：通常 `w_s = 0`，主要看 `q = delta h`。
- 节流阀：通常 `h1 = h2`。
- 绝热喷管：焓降转化为速度增量。
- 压气机：外界输入轴功，焓升高。
- 透平/动力机：输出轴功，焓降低。

## 第三章：气体和蒸汽性质

### 理想气体状态方程

理想气体模型假设分子是不占体积的弹性质点，且分子之间没有作用力。工程中常把空气、氮气、氧气、燃气、烟气等在常温常压附近近似为理想气体。

状态方程：

```text
p v = R_g T
p V = m R_g T
p V = n R T
R_g = R / M
```

### 理想气体比热容

定容比热 `c_v` 与定压比热 `c_p` 的关系：

```text
c_p - c_v = R_g
k = c_p / c_v
```

理想气体的内能和焓只与温度有关：

```text
du = c_v dT
dh = c_p dT
```

理想气体熵变常用式：

```text
ds = c_v dT/T + R_g dv/v
ds = c_p dT/T - R_g dp/p
```

若比热取定值：

```text
delta s = c_v ln(T2/T1) + R_g ln(v2/v1)
delta s = c_p ln(T2/T1) - R_g ln(p2/p1)
```

### 水蒸气

饱和状态是汽化速度与液化速度相等的动态平衡。饱和温度和饱和压力只有一个独立变量，压力升高时饱和温度升高。

基本术语：

- 饱和液：`t = t_s, p = p_s` 的液态水。
- 干饱和蒸汽：处于饱和状态的蒸汽。
- 湿饱和蒸汽：饱和液与干饱和蒸汽的混合物。
- 过热蒸汽：温度高于对应压力下饱和温度的蒸汽。
- 干度 `x`：湿蒸汽中干饱和蒸汽的质量分数。

水的定压加热汽化过程要记住“一点、两线、三区、五态”：

- 一点：临界点。
- 两线：饱和水线、干饱和蒸汽线。
- 三区：过冷水区、湿蒸汽区、过热蒸汽区。
- 五态：未饱和水、饱和水、湿蒸汽、干饱和蒸汽、过热蒸汽。

湿蒸汽区常用线性插值：

```text
y = y_f + x y_fg
```

其中 `y` 可以是 `v, u, h, s`。

## 第四章：气体和蒸汽基本热力过程

分析前提通常是理想气体、可逆过程。依据是理想气体状态方程、第一定律和熵变公式。

### 四个基本过程

| 过程 | 过程条件 | 状态关系 | 热量与功 |
| --- | --- | --- | --- |
| 定容 | `v = const` | `p2/p1 = T2/T1` | `w = 0`, `q = delta u = c_v delta T` |
| 定压 | `p = const` | `v2/v1 = T2/T1` | `w = p delta v`, `q = delta h = c_p delta T` |
| 定温 | `T = const` | `p1 v1 = p2 v2` | 理想气体 `delta u = delta h = 0`, `q = w` |
| 定熵 | `s = const` | `p v^k = const` | `q = 0`, 功由温度差决定 |

定熵关系：

```text
p v^k = const
T2/T1 = (v1/v2)^(k-1) = (p2/p1)^((k-1)/k)
```

### 多变过程

多变过程：

```text
p v^n = const
```

特殊情况：

- `n = 0`：定压。
- `n = 1`：定温。
- `n = k`：定熵。
- `n -> infinity`：定容。

多变过程功：

```text
w = (p2 v2 - p1 v1) / (1 - n)
```

技术功关系：

```text
w_t = n w
```

多变过程热量可写成：

```text
q = c_n (T2 - T1)
c_n = c_v (n - k) / (n - 1)
```

画图题要先判断压缩/膨胀、升温/降温、吸热/放热，再定位 p-v 与 T-s 图上的方向。

### 水蒸气基本过程

水蒸气不应随便套理想气体公式，优先查表或查图得到 `h, s, v, x`。

常用关系：

- 定容：`w = 0`，`q = delta u`。
- 定压：`w = p delta v`，`q = delta h`。
- 定温可逆：`q = integral T ds`。
- 定熵：`q = 0`，常用 h-s 图或蒸汽表判断终态。

## 第五章：热力学第二定律

第二定律的实质是热力过程具有方向性。它回答过程进行的方向、条件和限度。

两种表述：

- 克劳修斯说法：不可能把热从低温物体传到高温物体而不引起其他变化。
- 开尔文说法：不可能从单一热源取热并使之完全变为功而不引起其他变化。

### 卡诺循环与卡诺定理

卡诺循环由两个定温过程和两个绝热可逆过程组成，是两恒温热源间的理想可逆循环。

热机效率：

```text
eta_C = 1 - T_low / T_high
```

卡诺定理：

- 在相同高温热源和低温热源之间工作的一切可逆循环效率相等。
- 同温限间的不可逆循环效率必小于可逆循环效率。

### 克劳修斯不等式与熵

循环判据：

```text
integral(delta q / T_r) <= 0
```

- 等于 0：循环可行且可逆。
- 小于 0：循环可行但不可逆。
- 大于 0：循环不可行。

熵的定义来自可逆热量：

```text
ds = delta q_rev / T
delta s = integral(delta q_rev / T)
```

不可逆过程有：

```text
ds >= delta q / T
```

### 熵方程与熵增原理

闭口系：

```text
delta S = S_f,Q + S_g
```

其中 `S_f,Q` 是热量交换带来的熵流，`S_g` 是不可逆引起的熵产，且 `S_g >= 0`。

绝热闭口系：

```text
delta S = S_g >= 0
```

孤立系统熵增原理：

```text
delta S_iso >= 0
```

注意：熵增原理只适用于孤立系统。非孤立系统或孤立系统中的某个物体，熵可以增大、减小或不变。

常用熵变：

- 理想气体任意过程：用 `c_v ln(T2/T1) + R ln(v2/v1)` 或 `c_p ln(T2/T1) - R ln(p2/p1)`。
- 固体/液体定比热：`delta S = m c ln(T2/T1)`。
- 热源：`delta S = Q / T`。
- 功源与循环工质：熵变为 0。

### 㶲与损失

㶲是环境条件下能量中可转化为有用功的最高份额，也可理解为系统只与环境相互作用、可逆变化到环境平衡状态时所能做出的最大有用功。

能量可以分成：

```text
E = Ex + An
```

- 可无限转换的能量：电能、机械能、风能等，`Ex = E`。
- 不能转换的能量：环境热能，`Ex = 0`。
- 可有限转换的能量：高于环境温度的热能。

不可逆损失与熵产相连：

```text
I = T0 * delta S_gen
```

这条关系把第二定律从方向判断推进到“可用能损失”的量化。

## 第九章：气体动力循环

### 活塞内燃机理想循环

三个典型循环：

- 奥托循环：定容加热。
- 狄塞尔循环：定压加热。
- 萨巴德循环：混合加热。

常用参数：

```text
压缩比 epsilon = v1 / v2
定容增压比 lambda = p3 / p2
定压预胀比 rho = v4 / v3
```

比较循环时，必须说明比较条件。总复习课件强调：

- 压缩比相同、吸热量相同时：定容加热效率最高，混合加热居中，定压加热最低。
- 最高压力和最高温度相同时：比较结论会改变，不能机械套上一条。

### 布雷顿循环

燃气轮机定压加热理想循环由两个定压过程和两个定熵过程组成。

热效率：

```text
eta_t = 1 - 1 / pi^((k-1)/k)
```

其中 `pi` 为压比。理想布雷顿循环效率随压比增大而提高，但实际循环还受压气机效率、涡轮前温度、回热器与压损等限制。

提高燃气轮机热效率的措施：

- 回热。
- 在回热基础上分级压缩、级间冷却。
- 在回热基础上多级膨胀、中间再热。

注意：没有回热时，单纯级间冷却或再热不一定提高热效率。

## 第十章：蒸汽动力循环

简单蒸汽动力循环是朗肯循环，基本过程：

1. 汽轮机定熵膨胀。
2. 凝汽器定压放热。
3. 给水泵定熵压缩。
4. 锅炉定压吸热。

提高蒸汽动力循环热效率的途径：

- 提高平均吸热温度：提高初温、提高初压、再热。
- 降低平均放热温度：降低背压。
- 改变循环形式：回热、再热、联合循环、热电联产。

再热循环主要目的不是必然提高效率，而是提高乏汽干度；回热循环中不同支路质量不同，T-s 图面积不能直接代表热和功。

## 理想气体混合物与湿空气

理想气体混合物满足：

```text
p = sum p_i
V = sum V_i
```

前者是道尔顿分压力定律，后者是亚美格分体积定律。两者只适用于理想气体混合物。

湿空气题要把“干空气 + 水蒸气”作为混合物处理，常见关注点是分压力、含湿量、相对湿度、露点和焓。生活化判断如“湿润夏天水管上出现水珠”，本质是表面温度低于空气露点，水蒸气凝结。

## 作图题清单

必须能在 p-v 图与 T-s 图上表示：

- 定容、定压、定温、定熵、多变过程。
- 奥托、狄塞尔、萨巴德循环。
- 布雷顿循环。
- 朗肯循环及再热、回热的基本变化。
- 水蒸气定压汽化过程的一点、两线、三区、五态。

作图题的顺序：

1. 标初终态。
2. 判断过程约束。
3. 判断曲线斜率和方向。
4. 标热量和功的正负。
5. 用面积意义检查是否合理。

## 计算题模板

### 闭口系

1. 选系统。
2. 写 `q = delta u + w`。
3. 若可逆，补 `w = integral p dv`。
4. 若理想气体，写 `delta u = c_v delta T`。
5. 用过程方程或状态方程补齐未知量。

### 开口稳流

1. 选控制体。
2. 写 `q = delta h + delta(c_f^2)/2 + g delta z + w_s`。
3. 判断能否忽略动能、位能和热量。
4. 用典型设备关系简化：喷管、压气机、透平、换热器、节流阀。
5. 检查功和热的正负号。

### 第二定律

1. 先构造孤立系统或明确热源温度。
2. 写熵变：系统熵变、热源熵变、熵产。
3. 用 `delta S_iso >= 0` 判断可行性。
4. 用 `I = T0 S_gen` 计算㶲损。

## 最后 30 分钟检查清单

- 系统边界有没有选错。
- 闭口系用 `u`，开口稳流用 `h`。
- 功的正负号是否按“系统对外做功为正”。
- 热量正负号是否按“外界向系统传热为正”。
- 可逆体积功和技术功是否混淆。
- 节流过程是否记住 `h1 = h2`。
- 水蒸气题是否查表，而不是套理想气体。
- T-s 图面积只在同一质量流或同一工质口径下才可直接解释。
- 熵增原理是否只用于孤立系统。
- 卡诺效率温度必须用热力学温度。
- 循环比较是否写明“压缩比相同/吸热量相同/最高压力温度相同”等条件。
- 再热主要提高乏汽干度，回热主要提高平均吸热温度。

## 和项目能力的连接

工程热力学可以直接连接到飞行器动力方向：

- [[布雷顿循环]]：燃气涡轮发动机基础循环。
- [[标准大气模型]]：把飞行高度转化为入口压力、温度与密度。
- [[压气机喘振]]：热力循环外，还需要部件稳定性约束。

后续可以继续把每次作业题沉淀成“系统选择 - 方程 - 假设 - 结果检查”的例题节点。
