## 考试结构与提分顺序

两套期末卷呈现出稳定结构：8 道选择题、8 道填空题、3 道解答题。选择和填空主要考概念辨析、公式调用和一步计算；解答题常落在二维分布、随机变量函数、统计推断或极限定理。

最短提分路径：

1. 第 7 章抽样分布：先会写正态样本三件套。
2. 第 8 章参数估计：会判断 MLE、无偏/一致、置信区间。
3. 第 9 章假设检验：会分清单双侧、临界值、结论表述。
4. 第 3-5 章：练支持域、边缘密度、卷积、最大值、矩计算。
5. 第 6 章：切比雪夫、大数定律、CLT、样本量题。
6. 第 1-2 章：事件表达、CDF 合法性、常用分布参数和分位数方向。

## 每道计算题的固定流程

1. 写对象：事件、随机变量、参数、统计量分别是什么。
2. 写条件：独立吗，同分布吗，总体正态吗，方差已知吗，支持域是什么。
3. 选工具：条件概率、CDF、卷积、矩、CLT、卡方/t/F、MLE、置信区间或检验。
4. 查结果：概率是否在 [0, 1]，密度是否归一，自由度是否正确，结论是否与单双侧方向一致。

## 先背 12 条

- CDF 三条件：单调不减、右连续、两端极限为 0 和 1；连续型单点概率为 0。
- 标准化：若 `X ~ N(mu, sigma^2)`，则 `(X - mu) / sigma ~ N(0, 1)`。
- 方差捷径：`Var(X) = E(X^2) - (EX)^2`，`Var(aX + b) = a^2 Var(X)`。
- 协方差：`Cov(X,Y) = E(XY) - EX EY`；独立推出不相关，反向一般不成立。
- 指数分布无记忆：`P(X > s + t | X > s) = P(X > t)`。
- 泊松稀疏化：若 `N ~ P(lambda)`，每个个体独立保留概率 `p`，则保留数 `K ~ P(lambda p)`。
- 最大值：独立时 `F_max(z) = product F_i(z)`；iid 时 `F_max = F^n`。
- 正态样本三件套：`Xbar ~ N(mu, sigma^2/n)`，`(n-1)S^2/sigma^2 ~ chi^2(n-1)`，`Xbar` 与 `S^2` 独立。
- 未知方差检验均值用 t，自由度是 `n - 1`；已知方差用 z。
- 方差检验用卡方，两方差比检验用 F；拟合优度只拒绝右尾。
- 单侧临界值用 `1 - alpha`，双侧用 `1 - alpha/2`。
- 检验结论写“拒绝 / 不能拒绝 H0”，不要写“证明 H0 正确 / 错误”。

## 第 1-2 章：事件与一维分布

重点不是多背公式，而是把题面语言翻译对。

- “至少一个”优先用补事件。
- “至多一个”“恰有一个”先写事件表达式，再算概率。
- 无放回依次抽取若位置完全对称，直接用比例，不要枚举前面所有结果。
- 判断 CDF：单调、右连续、两端极限；二维 CDF 还要看矩形增量非负。
- 连续型变量 `P(X = a) = 0`，但离散型跳跃点概率等于 `F(a) - F(a-)`。
- 正态右尾比例题要看清“前 15%”还是“后 15%”：右尾 15% 对应第 85 百分位。

常用分布：

- Bernoulli(p)：`E = p`，`Var = pq`。
- B(n,p)：`E = np`，`Var = npq`；大样本可正态近似。
- P(lambda)：`E = Var = lambda`；独立泊松可加。
- Exp(lambda)：`F(x) = 1 - exp(-lambda x)`，有无记忆性。
- U(a,b)：`E = (a+b)/2`，`Var = (b-a)^2/12`。
- N(mu,sigma^2)：标准化后查标准正态分布。

## 第 3-4 章：二维分布与函数分布

二维题第一步永远是画支持域。若支持域不是直积区域，通常可以先否定独立性。

联合密度题模板：

1. 归一化求常数。
2. 固定 `x` 竖切求 `f_X(x)`，固定 `y` 横切求 `f_Y(y)`。
3. 条件密度写成联合密度除以边缘密度。
4. 独立性先看支持域，再比较 `f_X f_Y`。
5. 条件概率可写成区域积分之比。

函数分布题模板：

- 一维单调变换：反解 `x = g^{-1}(y)`，乘导数绝对值。
- 非一一变换如 `Y = X^2`：所有根的贡献都要加起来。
- `Z = X + Y`：独立时用卷积 `f_Z(z) = integral f_X(x) f_Y(z-x) dx`。
- `Z = X - Y`：写 `x = z + y`，积分上下限来自原支持域。
- 最大值：先求 CDF，`P(max Xi <= z)`；独立后才可拆成乘积。

## 第 5 章：数字特征

很多题不需要先求分布，直接用 LOTUS 和矩即可。

- LOTUS：`E[g(X)]` 直接对 `g(x)` 积分或求和。
- `Var(X+Y) = VarX + VarY + 2Cov(X,Y)`。
- `Cov(X,Y) = E(XY) - EX EY`。
- 独立可以推出 `E(XY)=EXEY`，但不相关一般不能推出独立。
- 求 `Var[g(X)]`：先算 `E[g(X)]`，再算 `E[g(X)^2]`。
- 常见陷阱：比较有偏估计量不能只比方差，要看 MSE。

高频二级结论：

- `Corr(X_i - Xbar, X_j - Xbar) = -1/(n-1)`。
- 正态总体下 `Var(S^2) = 2 sigma^4 / (n-1)`。
- `B^2 = (1/n) sum (Xi - Xbar)^2 = ((n-1)/n) S^2`，是方差的 MLE，有偏但一致。

## 第 6 章：大数、CLT 与样本量

先判断题目要“保证”还是“近似”。

- 切比雪夫是不依赖分布的保守保证：`P(|X-EX| >= eps) <= VarX / eps^2`。
- 大数定律：样本均值依概率收敛到期望；样本函数均值收敛到函数期望。
- CLT：总和标准化分母是 `sigma sqrt(n)`，样本均值标准化分母是 `sigma / sqrt(n)`。
- 二项正态近似：`(X - np) / sqrt(npq) approx N(0,1)`。
- 离散变量用连续正态近似时可考虑连续性修正，但按题目口径来。

切比雪夫给出的样本量通常远大于 CLT。看到“至少保证”偏切比雪夫；看到“近似”“充分大”偏 CLT。

## 第 7 章：抽样分布

正态总体样本题先写三件套：

```text
Xbar ~ N(mu, sigma^2/n)
(n-1)S^2 / sigma^2 ~ chi^2(n-1)
Xbar independent of S^2
```

常用构造：

- `Z ~ N(0,1)`，`U ~ chi^2(nu)` 独立，则 `Z / sqrt(U/nu) ~ t(nu)`。
- `U1 ~ chi^2(nu1)`，`U2 ~ chi^2(nu2)` 独立，则 `(U1/nu1)/(U2/nu2) ~ F(nu1,nu2)`。
- `T^2 ~ F(1, nu)`。
- 若 `F ~ F(nu1,nu2)`，则 `1/F ~ F(nu2,nu1)`。

最常见错误：把 t 的自由度写成 `n`，正确是 `n - 1`。

## 第 8 章：参数估计与置信区间

MLE 四步：

1. 写似然函数，同时写参数范围。
2. 取对数。
3. 求导或比较相邻值。
4. 检查边界和离散整数约束。

常用 MLE：

- Bernoulli / Binomial：`p_hat = Xbar` 或成功次数 / 总次数。
- Poisson：`lambda_hat = Xbar`。
- Exponential(lambda)：`lambda_hat = 1 / Xbar`。
- Normal：`mu_hat = Xbar`，`sigma_hat^2 = (1/n) sum (Xi - Xbar)^2`。

置信区间判断：

- 均值、方差已知：z。
- 均值、方差未知、正态总体：t，自由度 `n - 1`。
- 方差：卡方，注意左右端点分位数会倒置。
- 两总体均值差、方差未知但相等：合并方差 t。

解释口径：置信度是“重复抽样得到的区间覆盖固定参数的比例”，不要说“本次区间以 95% 概率包含参数”。

## 第 9 章：假设检验

检验题按这个顺序写：

1. 写 `H0` 和 `H1`，先由备择判断左侧、右侧还是双侧。
2. 判断方差是否已知，选择 z / t / chi-square / F。
3. 写统计量和在 `H0` 下的分布。
4. 写拒绝域或 p 值比较。
5. 用规范语言下结论。

口径：

- 右侧检验：统计量过大拒绝。
- 左侧检验：统计量过小拒绝。
- 双侧检验：绝对值过大或落在两端拒绝。
- 在 `alpha = 0.05` 拒绝，则在更宽松的 `alpha = 0.10` 也拒绝；反过来不一定。
- 未落入拒绝域写“不能拒绝 H0”，不要写“接受 H0”。

## 最后 30 分钟检查清单

- CDF 是否右连续，两端极限是否正确。
- 密度是否写支持域并积分为 1。
- 二维题是否先画区域，积分上下限是否来自区域。
- 非一一变换是否把多个根都加上。
- 卷积上下限是否同时满足两个原变量的支持域。
- `Var(X+Y)` 是否漏了协方差项。
- 总和与均值的 CLT 标准化分母是否混淆。
- t、卡方、F 的自由度是否写对。
- 置信区间方差端点的分位数顺序是否倒置。
- 假设检验是否按单双侧使用 `alpha` 或 `alpha/2`。
- 结论是否写成“拒绝 / 不能拒绝”，没有写“证明正确 / 接受”。
